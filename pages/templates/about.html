About Airline Passenger Satisfaction Predictor

This is a simple web application that use a machine learning model to predict the satisfaction of an airline passenger based on various features of the flight. The model was trained on the Airline Passenger Satisfaction dataset from Kaggle.

The model was trained using a Random Forest Classifier and achieved an accuracy of 0.95. The model was then deployed using Django and is hosted on PythonAnywhere.
